This is an experimental interface to Lagrange, and a work in progress. Please check back for updates and send comments to Rick Ree (rree at fieldmuseum dot org).

  1. Install required system software

    • Install Python (recommended version: 2.5)

    • Install scipy and numpy (note: must match version of Python)

  2. Download and unpack Lagrange (snapshot version {{=d}})

    • Linux, Mac: [tar archive]

    • Windows: [zip archive]

    • This will create a folder called Lagrange-{{=d}}

  3. Configure an analysis file (data, model, and tree)

    • [Click here to start the configurator]

    • Save it as: {name}.decmodel.py

  4. Run the analysis

    • Place {name}.decmodel.py in the Lagrange-{{=d}} folder

    • Run: python {name}.decmodel.py (Or in IDLE, choose Run:Run Module)

    • Results are stored in: {name}.results.txt
